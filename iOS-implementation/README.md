To install pods:

Download CocoaPods App\
Open application\
"File" -> "New Podfile From Xcode Project"\
Select .xcodeproj\
Add the following to the Podfile created:
```
pod 'Alamofire'
pod 'GoogleMaps'
pod 'GooglePlaces'
pod 'EAIntroView'
```
Click "install"

Pods have now been installed.

Use the new workspace project created for all coding.
